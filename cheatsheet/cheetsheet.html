Ruby Loops vs. JS Loops

Harry Park & Kenneth Geiler

3/5/2015

Here is our comparison of Ruby loops and JavaScript loops.

Ruby Loop

JS Loop

[1, 2, 3].each { |element| print "#{element}" }

=>1, 2, 3

var a = [1, 2, 3];
for (var i = 0; i < a.length; ++i) {
var element = a[i];
console.log(element);
}

=>1, 2, 3

Here are two loops: one in Ruby, one in JavaScript. As you can see, there is a difference in syntax but their functionality is the same. This is an example of difference of the structure of Ruby and JavaScript loops.
