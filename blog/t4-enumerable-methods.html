All about the cycle method!

Sunday, Feburary 8th, 2015

The Ruby method that I will be researching today is #cycle. After reading the Ruby Docs on this method, it seems that is it a powerful method that could be used in place of a while loop. Calling cycle on an array will infinitely print out the elements of the array. Also, if you definete a parameter for the method like .cycle(2), the result will be the array printed out twice. This can be very useful for different situations involving while loops. Cycle can even take advantage of the break command that stops an infinite loop at a certain point. An example of this is:
days_of_week.cycle {|day| puts day}
=> Monday
Tuesday
Wednesday
Thursday
Friday
Saturday
Sunday
Monday
Tuesday
...
